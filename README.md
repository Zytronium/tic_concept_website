# Tulsa Innovation Collective (TIC) Concept Website

While TIC is a fictional organization, this website demonstrates my abilities to create modern 
websites for similar clients in the real world. This website is based on a prototype for a real
cleint that decided to use fewer outside resources to build their website, leaving me with a
perfectly good template to rebrand, complete, and showcase on my website, webworks.zytronium.dev.

Nearly all content (but not structure and styling) on this website is AI generated. This is the
content I would normally have the client give me to fill out the website with.

Again, TIC is not a real organization, even though this website markets itself as if it were a real 
organization. If the Tulsa Innovation Collective does exist or ever exists in the future, it is mere
coincedence, and this website is not based on or impersonating that organization.
